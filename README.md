# Мобильный обходчик

MVP для кейса хакатона "ИТы Герой 2026": автоматизированная информационная
система "Мобильный обходчик - ТО оборудования энергоцеха".

## Цель

Подготовить мобильное PWA-приложение для оперативного персонала, которое
помогает выполнять обходы и осмотры оборудования на объекте без бумажных
маршрутных листов и ручного переноса данных в журналы.

## Технологический стек

- React 19
- Vite 8
- TypeScript
- React Router
- Redux Toolkit
- RTK Query
- Ant Design
- react-hook-form
- Zod
- vite-plugin-pwa

## Архитектура

Проект подготовлен под Feature-Sliced Design:

```text
src/
  app/        # инициализация приложения, провайдеры, router, store, theme
  pages/      # страницы приложения
  widgets/    # крупные композиционные блоки будущих экранов
  features/   # пользовательские сценарии
  entities/   # доменные модели и схемы
  shared/     # общие API, стили, инфраструктурные утилиты
```

## Команды

Запуск:

```bash
npm install
npm run build
npm run preview
```

Локальный dev-сервер:
