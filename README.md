# ИТы Герой

Демо MVP для хакатона "ИТы Герой 2026".

## PWA-тест фотофиксации

Сценарий сейчас один: открыть приложение на телефоне, дать доступ к камере,
сделать фото инцидента и увидеть получившийся снимок на экране.

```bash
npm run dev:host
```

После запуска откройте на телефоне адрес из блока `Network`, например
`http://192.168.1.20:5173`.

Для прямого доступа к `getUserMedia` и полноценной регистрации service worker
мобильный браузер обычно требует HTTPS. Если локальный HTTP не даст открыть
камеру, используйте кнопку "Системная камера" или запустите проект через
HTTPS-туннель до локального dev-сервера.

Перед проверкой production-режима:

```bash
npm run build
npm run preview:host
```
